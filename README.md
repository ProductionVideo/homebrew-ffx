# ProductionVideo Ffx

## How do I install these formulae?

`brew install productionvideo/ffx/<formula>`

Or `brew tap productionvideo/ffx` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "productionvideo/ffx"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
